## Overview

ORVX is the utility token that powers the Orvix network.

- **Total supply:** 1,000,000,000 ORVX (fixed)
- **Launch:** pump.fun

> Always verify the official token mint address via the project's verified
> Twitter / Discord. Never trust unverified sources.

## Utility

### Compute payments
ORVX is used to pay for inference. Billing logic is in place today; real on-chain
settlement is being finalized alongside live inference.

### Node staking
Providers stake to run nodes. Staking is **off-chain in v1**, moving **on-chain
in v2**.

### Premium access tiers
Holding ORVX unlocks tiered benefits, including pricing discounts:

| Tier    | Benefit |
|---------|---------|
| Bronze  | Base access |
| Silver  | Lower per-token pricing |
| Gold    | Greater discount + priority routing |
| Diamond | Maximum discount (up to ~25% off) |

### Buyback & burn
A portion of network revenue funds buyback-and-burn, reducing circulating
supply over time.

### Governance
Governance is **off-chain in v1**, moving **on-chain in v2**, giving the
community a say in network parameters and direction.

## Status

The token economy is early and subject to change as the network moves from
alpha toward public testnet. See [CHANGELOG.md](../CHANGELOG.md) for current
project status.
